## Getting started

### Prerequisites

You'll need npm and .NET installed on your system.

### Installation

1. clone the repo
   ```
   git clone https://github.com/DanteDeRuwe/piral-blazor-demo.git
   ```
2. install npm packages
   ```
   npm install
   ```
3. run the sample
   ```
   npx pilet debug
   ```
